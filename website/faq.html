FAQ

  • Q: One of the map sources by Google shows only red crosses instead of map tiles. What is wrong?
    A: Google changes the urls for their map tiles frequently. Please try to perform an online update (in settings dialog).
    If that doesn't solve the problem please make sure your IP is not banned by Google (can happen especially for the "Google Earth" map provider).
  • Q: My favorite online map is not in the list of maps provided by Mobile Atlas Creator - can it be added?
    A: Maps have to meets certain criteria for beeing compatible with Mobile Atlas Creator. A lot of maps have already been identified as incompatible. Please check this list if your favorite online is already known to be incompatible. The complete list can be found here.
  • Q: My favorite map/atlas format is not in the list of maps provided by Mobile Atlas Creator - can it be added?
    A: Most map formats can be added, but of course it is a bit of work and lots of information is needed for implementing a new atlas format. In general a feature request for a new atlas format hate to include the following information:
    1. Name of the atlas format and name of the application which uses this format (best with Web-link)
    2. Provide a detailed format description of the atlas format. If there is an online documentation available just link it in the feature request - otherwise you have to figure out the file format and describe it in the feature request.
    3. Attach at least one small example atlases in the desired output format to the feature request - including the information which region is covered by the example(s)
    BTW: If any atlas format supported by MOBAC is similar to the desired atlas format please name it - this helps a lot and saves time.
