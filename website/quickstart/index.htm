Welcome This is a quick start manual for the Mobile Atlas Creator that will help you get started with this useful application. About Mobile Atlas Creator Have you ever been frustrated because the map or navigation application on your phone would take forever to load? Mobile Atlas Creator (formerly known as TrekBuddy Atlas Creator) is an open source (GPL) program which creates offline atlases for GPS handhelds and cell phone applications like TrekBuddy, AndNav and other Android and WindowsCE based applications. Individual maps can be exported as one large PNG image with calibration MAP file for OziExplorer. As source for an offline atlas Mobile Atlas Creator can use a large number of different online maps such as OpenStreetMap and other online map providers. Purpose of this Quick Start Manual As fervent users of Mobile Atlas Creator, my partner and I agreed that a more user friendly document would be helpful to a wider range of people that want to start using the program. Thus this web page is created in order to make it easier for future users start using this program. Requirements needed to install the program, installation process, how to use the program once it is installed, and how to deal with the settings are explained in a straightforward manner. © Mobile Atlas Creator 2012
